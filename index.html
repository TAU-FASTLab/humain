HUMAIN

Human-Machine Interaction Modeling and validation

Introduction

This tool, the Human-Machine interaction modelling and validation (HUMAIN) tool enables designers and developers to validate that their interfaces cover all those aspects relevant to support an intuitive and personalised interaction between humans and machines/robots. This tool is especially intended for developers and UI/UX designers in the early stages of front end development.

With this tool, you can fill out a simple questionnaire about the UI/UX elements of a software application you're developing and get feedback automatically.

Here's how you use the tool:

  1. Press the start button below
  2. Fill out the questionnaire that appears
  3. Press the "submit" button below the bottom of the questionnaire. Recommendations should appear shortly

Start the questionnaire by pressing the start button below.

About the tool

This tool was collaboratively developed between TECNALIA research center and Tampere University under the research initiatives of the SHOP4CF European project.

Scope

The Human-Machine interaction modelling and validation (HUMAIN) tool enables designers and developers to validate that their interfaces cover all those aspects relevant to support an intuitive and personalised interaction between humans and machines/robots.

Through a set of questions dealing with user experience (UX) in human-machine interactions (HMI) with special focus on the end-users, this tools provides a set of recommendations (based on interaction patterns and design guidelines) that designers and developers should implement in their interfaces to ensure usability, accessibility and UX aspects.

The questions cover relevant aspects about the human-machine interactions, such as the type of devices to be used (e.g. tablet, PC), the type of technology (e.g. Augmented Reality, Web) and the target user profile. If you want to use HUMAIN tool, you can press the start button below.

About this tab

Below, you will see recommendations for your application based on your answers to the questionnaire. The feedback is broken into sections based on the categories of the different UI/UX aspects. Feel free to go back to the home page to double-check your answers if you feel the need to do so.

Simply click on the arrows next to each section's title to open them. The "minimum" recommendation means the basic goal your software should achieve regarding the relevant aspect, while the "maximum" recommendation is more ideal.

Please fill out the questionnaire and click "submit" before entering this tab.
